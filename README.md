# python-nuxt-starter

## Backend

#### Install / Prepare the Backend

```bash
cd backend
python -m venv .env
source .env/bin/activate
pip install -r requirements.txt
```

#### Run the Backend

```bash
python app.py
```

## Frontend

<small>See also [./frontend/README.md](./frontend/README.md)</small>

#### Install / Prepare the Frontend

```
cd frontend
npm install
```

#### Run The Frontent

```
npm run dev
```
